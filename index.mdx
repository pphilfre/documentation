Markup is a fast, keyboard-first markdown workspace built with Next.js, TypeScript, Convex, WorkOS AuthKit, and shadcn/ui. It combines a powerful markdown editor with advanced note tooling — split preview, graph/backlink view, whiteboards, mind maps, sharing, and local/remote sync — and supports both web and desktop (Tauri) targets.

## Key highlights:

- Keyboard-first markdown editing with live preview (GFM, math via KaTeX, syntax highlighting, Mermaid).
- Multiple workspace modes: Editor, Split, Preview, Graph, Whiteboard, Mind Map.
- File & folder organisation (colours, hierarchy, pinning, tags).
- Cloud sync via Convex \+ offline-aware local-first edits.
- Authentication with WorkOS AuthKit and optional desktop support via Tauri.
